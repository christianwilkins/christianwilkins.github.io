# Promotion Using Existing Accounts

## LinkedIn
- Add the site to the **Featured** section.
- Post a short “what I build” thread with:
  - A 1–2 sentence positioning statement (software consultant + product design + AI workflows).
  - 2–3 project highlights with outcomes/impact.
  - A direct link to the site.

## GitHub
- Pin the portfolio repo on your profile.
- Add the site link to your profile README (top section).

## X / Twitter
- Pin a short launch thread with 2–3 project highlights.
- Include a direct link to the site in the first post.

## Email Signature
- Include the site link and a **Book a Call** link.
- Example format:
  - Christian Wilkins
  - Software Consultant & Engineer
  - https://chriswiki.com
  - Book a Call: https://cal.com/christianwilkins/book?duration=15
