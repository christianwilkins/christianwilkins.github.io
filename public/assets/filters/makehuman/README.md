# Filters Studio MakeHuman assets

The female proxy mesh, hair meshes, and skin textures in this directory are from the MakeHuman Community asset repository:

- Source: https://github.com/makehumancommunity/makehuman-assets
- License: CC0 1.0 Universal
- License text: `LICENSE.txt`

The browser renderer uses the assets as OBJ meshes and PNG textures, with Three.js applying the interactive proportion transforms and materials.
